## Embed Zones

The site includes clearly marked embed zones for connecting external applications (Streamlit, dashboards, etc.). Search for EMBED_APP_1_HERE, EMBED_APP_2_HERE, or EMBED_APP_3_HERE in the HTML to locate insertion points.

To embed an app, replace the placeholder content with an iframe:

<iframe src="https://your-app.streamlit.app" width="100%" height="600" frameborder="0" style="border:none;"></iframe>
